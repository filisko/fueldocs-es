Troubleshooting

This page will try to list some common problems, and how to avoid or fix them.

Using Google Mail

You need to use the following configuration:

  1. Use 'smtp' as driver.
  2. The SMTP host should be "ssl://smtp.gmail.com".
  3. The SMTP port should be 465
  4. You should set 'newline' to "\r\n".
